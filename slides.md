### Coding

>Write a function that reverse a string!

---

```c++
    string reverse(string str){
        int left = 0;
        int right = str.size() - 1;
        while(left < right) {
            char temp = str[left];
            str[left] = str[right];
            str[right] = temp;
            left++;
            right--;
        }
        return str;
    }
```
